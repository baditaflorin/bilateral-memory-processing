# Bilateral Memory Processing

Private browser-based EMDR-inspired audio journaling with local transcription, reflection prompts, and no trauma uploads.

Live site: https://baditaflorin.github.io/bilateral-memory-processing/

Repository: https://github.com/baditaflorin/bilateral-memory-processing

Support: https://www.paypal.com/paypalme/florinbadita

## Status

Initial scaffold is published first so GitHub Pages works from day one. The full local-only application is implemented in follow-up commits.

## Quickstart

```sh
npm install
make dev
make build
make test
make smoke
```
