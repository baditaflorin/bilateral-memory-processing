# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, safe, and welcoming.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different experiences and viewpoints.
- Give and receive constructive feedback gracefully.
- Prioritize user safety and privacy.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Sharing another person's private information without consent.
- Uploading or committing real sensitive mental health content.
- Presenting the app as clinical treatment or emergency care.

## Enforcement

Maintainers may remove comments, commits, issues, or pull requests that violate this code of conduct.
