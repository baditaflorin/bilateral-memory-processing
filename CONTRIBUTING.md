# Contributing

Thanks for helping improve Bilateral Memory Processing.

## Local workflow

1. Install dependencies with `npm install`.
2. Install local hooks with `make install-hooks`.
3. Run `make fmt`, `make lint`, `make test`, and `make smoke` before pushing.
4. Use Conventional Commits for commit messages, such as `feat: add session timer`.

## Safety and scope

This project is a private journaling and grounding aid. It is not a medical device and must not be presented as a replacement for licensed mental health care.

Do not commit recordings, transcripts, real user trauma material, secrets, or private model files.
